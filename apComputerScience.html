Should you take the Class?

Yes: 80% No: 20%

Why You Should Take this Class...

- This class teaches you a lot of important skills involved with programming- either in JAVA code which is a very commonly used code or programming HTML & CSS to create websites. These are real-life skills you can continue learning and get a very goood job from.
- This class has super fun projects, that you can choose to work with a partner or work solo.
- This class has a super nice teacher that you can go to if you have any problems.

Why You Shouldn't Take this Class...

- This class is very much based on your schedule, in that you can do a lot of work whenever you want. This can be a good thing, but for many this causes problems due to procrastination. This class gets very hard if you procrastinate a lot.
- This class has a lot of independent learning, that can be difficult for many people.
